# Apple Clone - React Website

A stunning, pixel-perfect clone of Apple.com built with React, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Authentication System**: Mock login with protected routes
- **Responsive Design**: Mobile-first approach with smooth animations
- **Product Showcases**: Individual pages for iPhone, iPad, Watch, TV, AirPods, and Mac
- **Apple Services**: Complete showcase of Apple's service ecosystem
- **Real Apple Branding**: Authentic Apple logo and product imagery
- **Modern UI/UX**: Apple's signature clean design with hover effects and transitions

## 🛠️ Technologies Used

- **React 18** with TypeScript
- **React Router** for navigation
- **Tailwind CSS** for styling
- **Vite** for build tooling
- **Lucide React** for icons

## 📱 Pages

- `/login` - Authentication page
- `/home` - Homepage with hero slider and services
- `/iphone` - iPhone product showcase
- `/ipad` - iPad product showcase  
- `/watch` - Apple Watch showcase
- `/tv` - Apple TV showcase
- `/airpods` - AirPods showcase
- `/mac` - Mac product showcase

## 🚀 Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/apple-clone.git
cd apple-clone
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Building for Production

```bash
npm run build
```
